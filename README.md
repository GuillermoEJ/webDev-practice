

# User Manager - MERN Stack

This application is built using the industry-standard MERN stack:

- **MongoDB:** NoSQL database used to store user documents in flexible, JSON-like formats.
- **Express.js:** Minimal and flexible Node.js web application framework used to build our RESTful API.
- **React:** Component-based frontend library (v18) used for building interactive user interfaces.
- **Node.js:** JavaScript runtime environment that executes the backend application.

*Additional tools:* `Mongoose` (ODM for MongoDB), `Axios` (HTTP client), `React-Bootstrap` (UI framework), and `React-Router-Dom` (Client-side routing).

## Features

- **Full CRUD Operations:** Create, Read, Update, and Delete data seamlessly.
- **RESTful API:** Structured backend with Express.js managing modular HTTP routes.
- **State Management:** Fully dynamic UI leveraging React Hooks (`useState`, `useEffect`).
- **Component-Based UI:** Modular front-end logic separating concerns (Modals, Lists, Forms).
- **Client-Side Routing:** Fast, single-page application navigation using React Router.
- **Data Modeling:** Robust database schemas enforced by Mongoose.
